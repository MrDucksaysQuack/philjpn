## 🎯 대시보드 통합 계획

### Phase 1: 메인 페이지 대시보드화 (우선순위: 최고)

**구현 내용**:

```typescript
// app/page.tsx 구조
if (user) {
  // 대시보드 표시
  return <UserDashboard />
} else {
  // 기존 Hero 섹션
  return <LandingPage />
}
```

**대시보드 구성 요소**:

1. **빠른 통계 카드 (4개)**
   - 오늘 응시한 시험 수
   - 이번 주 목표 진행률
   - 현재 활성 목표 수
   - 최근 학습 패턴 점수

2. **목표 진행 상황 위젯**
   - 가장 중요한 목표 1개 표시
   - 진행률 바
   - 남은 기간
   - "목표 설정하기" 버튼 (목표 없을 때)
   - "자세히 보기" → `/analysis?tab=goals`

3. **추천 시험 위젯**
   - 상위 3개 추천 시험 카드
   - 적합도 점수 표시
   - "모든 추천 시험 보기" → `/exams/recommended`

4. **최근 학습 활동 위젯**
   - 최근 3개 시험 결과
   - 점수, 개선 추이
   - "전체 결과 보기" → `/results`

5. **학습 패턴 인사이트 위젯**
   - 최적 학습 시간대
   - 주요 약점 영역 (상위 3개)
   - 효율성 지표 요약
   - "자세한 분석 보기" → `/analysis`
