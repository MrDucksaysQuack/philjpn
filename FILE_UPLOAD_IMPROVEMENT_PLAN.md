# 파일 업로드 기능 개선 계획

## 현재 상황 분석

### 사용 위치
1. **BasicInfoTab**: 로고, 파비콘
   - 현재: 파일 업로드 + URL 입력 둘 다 지원
   - 문제: URL 입력이 복잡하고 불편함

2. **TeamTab**: 팀원 프로필 이미지
   - 현재: URL 입력만 지원
   - 문제: 파일 업로드 불가능

3. **MarkdownEditor**: 이미지 삽입
   - 현재: URL 수동 입력
   - 문제: 이미지 업로드 불가능

### 현재 문제점
1. ❌ URL 입력 방식이 일반 사용자에게 불편함
2. ❌ 파일 직접 업로드가 일관되지 않음
3. ❌ 드래그 앤 드롭 미지원
4. ❌ 업로드 진행률 미표시
5. ❌ MediaUploader 컴포넌트가 하드코딩된 색상 사용

## 개선 방안

### 🎯 우선순위 높음 (즉시 개선)

#### 1. **통합 파일 업로드 컴포넌트 생성**
- URL 입력 필드 완전 제거
- 파일 직접 업로드만 지원
- 드래그 앤 드롭 지원
- 업로드 진행률 표시
- 미리보기 개선 (이미지 썸네일, 삭제 버튼)
- 테마 색상 적용

#### 2. **BasicInfoTab 개선**
- URL 입력 필드 제거
- 파일 업로드만 지원
- 드래그 앤 드롭 추가

#### 3. **TeamTab 개선**
- URL 입력을 파일 업로드로 변경
- 팀원 프로필 이미지 업로드 지원

#### 4. **MarkdownEditor 개선**
- 이미지 업로드 버튼 클릭 시 파일 선택
- 업로드 후 마크다운 자동 삽입

### 🎯 우선순위 중간 (단계적 개선)

#### 5. **이미지 편집 기능**
- 이미지 크롭
- 이미지 리사이즈
- 회전 기능

#### 6. **다중 파일 업로드**
- 여러 이미지 한 번에 업로드
- 갤러리 형태 미리보기

#### 7. **파일 관리**
- 업로드된 파일 목록
- 파일 삭제 기능
- 파일 재사용

## 구현 계획

3. ✅ 다국어 지원

### Phase 2: BasicInfoTab 개선
4. ✅ URL 입력 필드 제거
5. ✅ FileUploader 적용

### Phase 3: TeamTab 개선
6. ✅ 프로필 이미지 업로드로 변경

### Phase 4: MarkdownEditor 개선
7. ✅ 이미지 업로드 기능 추가
