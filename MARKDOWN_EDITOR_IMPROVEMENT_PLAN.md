# 마크다운 에디터 개선 계획

## 현재 상태 분석

### 사용 위치
- **CompanyTab**: `aboutCompany` (회사 소개 마크다운)
- **TeamTab**: `aboutTeam` (팀 소개 마크다운)
- **ServiceTab**: `serviceInfo` (서비스 소개 마크다운)

### 현재 문제점
1. ❌ 기본적인 정규식 기반 파싱만 지원 (제한적)
2. ❌ 편집/미리보기 탭 전환 방식 (불편함)
3. ❌ 하드코딩된 한국어 텍스트
4. ❌ 테마 색상 미적용
5. ❌ 툴바 없음 (수동 입력만 가능)
6. ❌ 이미지 업로드 미지원
7. ❌ 문법 하이라이팅 없음
8. ❌ 단축키 미지원

## 개선 방안

### 🎯 우선순위 높음 (즉시 개선)

#### 1. **Split View 모드 추가**
- 편집과 미리보기를 동시에 보기
- 좌우 분할 또는 상하 분할 옵션
- 실시간 미리보기 업데이트

#### 2. **툴바 추가 (WYSIWYG 스타일)**
```
[ B ] [ I ] [ H1 ] [ H2 ] [ H3 ] [ 링크 ] [ 이미지 ] [ 리스트 ] [ 코드 ]
```
- 버튼 클릭으로 마크다운 문법 자동 삽입
- 선택된 텍스트에 자동 적용
- 커서 위치에 문법 삽입

#### 3. **react-markdown 라이브러리 사용**
- 현재: 간단한 정규식 파싱
- 개선: 전문적인 마크다운 파싱 라이브러리
- 더 정확한 렌더링 및 다양한 문법 지원

#### 4. **다국어 지원**
- 하드코딩된 텍스트를 번역 키로 변경
- 언어별 안내 메시지

#### 5. **테마 색상 적용**
- 하드코딩된 `gray-300`, `gray-50` 등을 테마 클래스로 변경
- 동적 테마 지원

### 🎯 우선순위 중간 (단계적 개선)

#### 6. **이미지 업로드 지원**
- 드래그 앤 드롭으로 이미지 업로드
- 업로드된 이미지 URL 자동 삽입
- 미리보기에서 이미지 표시

#### 7. **문법 하이라이팅**
- 코드 에디터처럼 마크다운 문법 강조
- 실시간 문법 검증

#### 8. **단축키 지원**
- `Ctrl+B`: 볼드
- `Ctrl+I`: 이탤릭
- `Ctrl+K`: 링크
- `Ctrl+Shift+I`: 이미지

#### 9. **템플릿 제공**
- 자주 사용하는 마크다운 패턴
- 예: "회사 소개 템플릿", "팀 소개 템플릿"

### 🎯 우선순위 낮음 (선택적 개선)

#### 10. **자동완성**
- 링크 자동완성
- 이미지 URL 자동완성

#### 11. **문자 수 카운터**
- 현재 입력된 글자 수 표시
- 권장 글자 수 안내

#### 12. **버전 히스토리**
- 마크다운 편집 히스토리
- 되돌리기/다시하기
