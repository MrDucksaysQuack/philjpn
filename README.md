# Exam Platform

시험 관리 및 학습 플랫폼 프로젝트

## 📋 프로젝트 개요

이 프로젝트는 시험 생성, 관리, 응시 및 결과 분석을 위한 종합 플랫폼입니다.

## 🏗️ 프로젝트 구조

```
exam-platform/
├── backend/          # NestJS 백엔드
├── frontend/         # Next.js 프론트엔드
└── docs/            # 프로젝트 문서
```

## 🚀 시작하기

### 백엔드 실행
```bash
cd backend
npm install
npm run start:dev
```

### 프론트엔드 실행
```bash
cd frontend/client
npm install
npm run dev
```
